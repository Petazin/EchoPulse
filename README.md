# EchoPulse

EchoPulse is a World of Warcraft addon that allows you to send periodic messages to different chat channels automatically.

## Features

* **Graphical Interface**: Manage all your messages in the Interface Options menu.
* **Multi-Message**: Add as many automated messages as you need.
* **Independent Timers**: Each message has its own frequency and state.
* **Hyperlinks Support**: Supports items, spells, crafts, achievements...

## Commands

* `/pulse [on|off]` : Start / stops sending all enabled messages.
* `/pulseconfig` : Opens the graphical configuration panel.
* `/pulsemsg <message>` : Sets the message for the first entry (backward compatibility).
* `/pulseinfo` : Displays the status of all entries.
* `/pulsehelp` : Displays help message.

The settings are saved per character/realm.

## EchoPulse (Español)

EchoPulse es un addon para World of Warcraft que permite enviar mensajes periódicos a diferentes canales de chat de forma automática.

## Características

* **Interfaz Gráfica**: Gestiona todos tus mensajes en el menú de Opciones de Interfaz.
* **Mensajes Múltiples**: Añade tantos mensajes automáticos como necesites.
* **Temporizadores Independientes**: Cada mensaje tiene su propia frecuencia y estado.
* **Soporte de Hipervínculos**: Soporta objetos, hechizos, profesiones, logros...

## Comandos

* `/pulse [on|off]` : Inicia / detiene el envío de todos los mensajes habilitados.
* `/pulseconfig` : Abre el panel de configuración gráfica.
* `/pulsemsg <mensaje>` : Establece el mensaje para la primera entrada (compatibilidad).
* `/pulseinfo` : Muestra el estado de todas las entradas.
* `/pulsehelp` : Muestra el mensaje de ayuda.

Los ajustes se guardan por personaje/reino.
